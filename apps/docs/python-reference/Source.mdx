## Source

```python
class Source(BaseSource)
```

Representation of a source, which is a collection of documents and passages.

**Arguments**:

- `id` _str_ - The ID of the source
- `name` _str_ - The name of the source.
- `embedding_config` _EmbeddingConfig_ - The embedding configuration used by the source.
- `created_at` _datetime_ - The creation date of the source.
- `user_id` _str_ - The ID of the user that created the source.
- `metadata_` _dict_ - Metadata associated with the source.
- `description` _str_ - The description of the source.
