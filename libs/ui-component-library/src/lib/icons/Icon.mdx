# Creating a new icon

To create a new icon, follow these steps:

1. Create a new SVG file in the `libs/ui-component-library/src/lib/icons/sources` directory. The file should be named after the icon in a dash format, e.g., `my-icon.svg`.
2. Run `npm run cl:generate-icons` to generate the new icon component.

# Icon Sources

We get our icons from
https://fonts.google.com/icons?icon.query=folder&icon.size=24&icon.color=%235f6368

Export icons at 20x20px, make sure to use the "sharp" style to get hard corners (optical size = 20, grade = 0, weight = 400). Once you have the SVG, make sure to set the fill to currentColor (it may get set to a hardcoded value during the download).

Make sure you export fonts at size 16px
