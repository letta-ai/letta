# Panel
Panels are used to manage the layout of the Agent Development Environment. They can be moved, resized, and docked to different areas of the screen.

## Usage


<div className="flex flex-col gap-4">
  <PanelManager>

          Panel Content
        </div>
      </Panel>

      <Panel id={['test', 'data']} trigger={<Button label="Activate The subpanel" />} title="Sub panel 1">
        <div style={{ padding: '1rem' }}>

      <Panel id={['different']} trigger={<Button label="Activate Another panel" />} title="Main panel 2">
