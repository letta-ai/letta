# Creating a new icon

To create a new icon, follow these steps:

1. Create a new SVG file in the `libs/component-library/src/lib/icons/sources` directory. The file should be named after the icon in a dash format, e.g., `my-icon.svg`.
2. Run `npm run cl:generate-icons` to generate the new icon component.
