# All about Forms
We use forms to collect user input. We use [react-hook-form](https://react-hook-form.com/) to manage form state and validation.

## Usage
Below is an example of how to use the Form component to create a form with a username and email field.

```tsx
import { FormProvider, FormField, Input, useForm } from '@letta-web/component-library';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';

const exampleSchema = z.object({
  name: z.string(),
  email: z.string().email(),
});


export const ExampleForm = () => {
  const form = useForm({
    resolver: zodResolver(exampleSchema),
  });

  const onSubmit = (data) => {
    console.log(data);
  };

  return (
    <FormProvider {...form}>
      <form onSubmit={form.handleSubmit(onSubmit)} className="w-2/3 space-y-6">
         <FormField
          control={form.control}
          name="username"
          render={({ field }) => (
            <Input label="Username" {...field} />
          )}
        />
        <FormField
          control={form.control}
          name="email"
          render={({ field }) => (
            <Input label="Email" {...field} />
          )}
        />
        <button type="submit">Submit</button>
      </form>
    </FormProvider>
  );
};
```
